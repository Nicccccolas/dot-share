# Dot Share Server

## Installation

1.  Clone repository in your favourite local direction and go to this direction

```bash
git clone https://github.com/Nicccccolas/dot-share.git

cd dot-share
```

2. Install all dependencies

```bash
npm install
```

3. Run Server

```bash
npm run dev
```

## Stack

- Typescript
- Express
- Postgres
